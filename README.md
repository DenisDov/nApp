# nApp

nApp is a personal React Native Sandbox project.

## Installation

```bash
yarn install
cd ios && pod install && cd ..
```

## How to run:

#### Android:

```
yarn android:dev
yarn android:prod
```

#### IOS:

##### Running on simulator/real device:

```
yarn ios:dev
yarn ios:prod
```

##### or

1. Choose scheme;
1. Run on simulator/device;

##### Assembling build (release mode):

1. Choose scheme;
1. Run "Product/Archive";
